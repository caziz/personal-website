BruinMeet

BruinMeet

Node.js, React, Swift, React Native

BruinMeet is a fun dating website and app exclusively for UCLA students. Instead of swiping forever, BruinMeet aims to quickly connect Bruins with their ideal and genuine match. Users can fill out their personal profile and answer fun questions to allow the app to optimize their match. BruinMeet focuses on encouraging its users to actually meet instead of maximizing time on the app. BruinMeet has a built-in date picker that sets up dates for the user which minimizes awkwardness and allows for the beginning of a genuine relationship.

Since BruinMeet's launch, over 500 Bruins have joined the app as active users.

SEASide brings the trouble of SEASNET to sea level!
SEASide

SEASide

Python, Kivy, Paramiko

SEASide is the best way to use the SEASNET linux servers at UCLA. SEASide provides a free cross platform graphical user interface that allows users to login to SEASNET. SEASide saves your username and password as well as automates the process, saving you from monotonous and unintuitive commands. SEASide creates tabs for each connection to the server and uses your custom server settings to automate the process. For each connection a file explorer is displayed to streamline navigation on the server. In addition, a terminal is emulated that securely streams commands entered by the user to the server. The terminal also displays text and special characters that are received as well as nontrivial applications like emacs and vim.

Behind the scenes, SEASide handles many techincal difficulties including automatic SSH authentication and SFTP file exploration. The terminal is also accurately emulated using cross platform libararies to maximize accessibility to the UCLA community. For our efforts, we won the First Place Overall Award at Hack on the Hill IV.

Catch up on news in minutes!
OneThreeFive

OneThreeFive

Swift, Firebase, Core Data

Catch up on the news throughout your busy day with the OneThreeFive app. It's simple: tap the 'One', 'Three', or 'Five' button to get a news article that takes that many minutes to read. Update your preferences with your favorite news sources to filter what type of articles you will receive. Favorite articles to save them to read again later. Stay informed in just minutes!

OneThreeFive uses a Firebase database to store the articles. Top headlines are analyzed for reading time and then sorted. Internally, the app requests an article with a user's specified preferences and prefered reading time. The user is then sent a trending article that the user has not read yet. The ReadabilityKit, SwiftyJSON, and News APIs are used to fetch and analyze news articles. Alamofire is used to communicate to the Firebase server. Core Data is used to save preferences locally.

Civic participation made easy
ReachOut

ReachOut

React, Firebase, D3

ReachOut is a website to find your representatives at all levels of government and share template messages to make contacting them as easy as possible. Given an address, the site lists politicians who have some responsibility for that jurisdiction, as well as multiple ways to reach them. Furthermore, ReachOut provides an editor for users to create template messages that they can then be shared with a short link. Users can browse templates created by others and a fully-populated email requires a single click.

The front-end is built with React. The site is hosted on Firebase behind a domain from Domain.com, and it uses a Firebase database to store the template messages. Political representation data is fetched from Google's Civic Information API. The user's address is automatically determined using native geolocation APIs combined with Google Maps Geocoding.

Don't just blindly throw darts at a dartboard - let us plan your trip for you.
Dartboard

Dartboard

Flask, jQuery, Jinja2

Dartboard is a website that aims to generate your ideal trip made especially for you. Just enter the location where you will be staying, the duration of your stay, and how long you want to spend touring each day. The site will then calculate the best places to visit every day for your entire trip, accounting for attraction ratings and travel time.

Dartboard starts with a simple search or a selection from the most popular destinations and ends with a cleverly designed itinerary equipped with links to Bing Maps directions and Yelp reviews for each event of your trip. Finally, planning to travel to a foreign place is simple and stress-free. Dartboard utilizes Flask, Bootstrap, Jinja2, jQuery, Bing Maps API and Yelp API.

What the Happening

Node.js, AngularJS, SQLite

What the Happening is a web app that allows users to input and view popular events in a given area. Each event's time and description are displayed in chronological order on the left sidebar. Quickly find as many words as possible
Boggle

Boggle

Java, Swing

Boggle recreates a virtual version of the classic dice game where players try to find words in sequences of adjacent letters in a limited amount of time. Compete to earn the highest score before the timer runs out. Users can customize the board size, name, and timer to match their preferences. The player can choose to either click on the interactive board or watch the letters become highlighted as they type thier words themselves. The application uses Java swing libraries for the GUI and color graphics.
